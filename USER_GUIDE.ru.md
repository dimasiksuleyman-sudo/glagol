# Руководство пользователя — Глагол

Глагол превращает длинные русские тексты в аудио. Вставляете текст или загружаете файл — получаете озвучку профессиональным голосом, которую можно слушать где угодно: на прогулке, в дороге, за домашними делами.

Создан для тех, кто предпочитает слушать, а не читать с экрана. Если вы любите аудиокниги, но хотите слушать *свои* документы — статьи, договоры, книги в PDF — Глагол для этого и сделан.

---

## Установка

1. Скачайте `Glagol_0.1.0_x64-setup.exe` со [страницы релизов](../../releases).
2. Запустите установщик.
3. Windows покажет предупреждение SmartScreen («Windows защитила ваш компьютер»). Это нормально для новых приложений без коммерческого сертификата подписи. Нажмите **«Подробнее»** → **«Выполнить в любом случае»**.
4. Готово — Глагол запустится автоматически.

**Что нужно:**
- Windows 10 или 11 (64-бит)
- ~26 МБ на диске
- Бесплатный ключ SaluteSpeech от Сбера (как получить — ниже)
- Интернет (озвучка работает через облако Сбера)

---

## Настройка ключа SaluteSpeech

Глагол использует синтез речи SaluteSpeech от Сбера. Чтобы он заработал, нужен бесплатный ключ.

1. Зайдите на [developers.sber.ru](https://developers.sber.ru), создайте проект **SaluteSpeech API**.
2. Скопируйте готовую строку **Authorization Key** (это Base64 от `client_id:client_secret`).
3. В Глаголе откройте **Настройки** → вставьте ключ в поле **SaluteSpeech Authorization Key** → **Сохранить**.
4. Нажмите **Проверить** — статус должен стать «подтверждён Сбером».

Ключ хранится локально, в Windows Credential Manager. Он никуда не передаётся, кроме самого Сбера при озвучке.

![Настройки](docs/screenshots/settings-page.png)

**Бесплатный тариф Сбера:** 200 000 символов синтеза в месяц. Счётчик в Настройках показывает, сколько осталось (ориентировочно. Точные цифры в личном кабинете на [developers.sber.ru](https://developers.sber.ru)).

## Первая озвучка

1. Откройте **Озвучить**.
2. Вставьте текст в поле — или нажмите **«Выбрать файл»** и загрузите документ.
3. Выберите голос.
4. Нажмите **«Озвучить и сохранить в библиотеку»**.

Через несколько секунд готовая озвучка появится в Библиотеке.

![Озвучить](docs/screenshots/synthesize-page.png)

**Поддерживаемые форматы файлов:** `.txt`, `.md`, `.docx`, `.pdf`.

**Голоса (6 штук):** Наталья, Борис, Марфа, Тарас, Александра, Сергей. У каждого своя манера — попробуйте разные, выберите тот, что приятнее слушать.

**Язык:** Глагол сделан для **русского текста**. Латиница и другие языки озвучиваются «на любителя».

**Объём:** комфортно озвучивает несколько тысяч символов за раз. Большие документы (книги, длинные PDF) обрабатываются целиком — текст автоматически разбивается на части.

## Библиотека

Все ваши озвучки хранятся в Библиотеке. Здесь можно:

- **▶ Слушать** — встроенный плеер с перемоткой
- **✏ Переименовать** — нажмите карандаш, введите новое название
- **⬇ Скачать** — сохранить WAV-файл куда угодно
- **🗑 Удалить** — убрать из библиотеки

![Библиотека](docs/screenshots/library-page.png)

Документы отсортированы от новых к старым. У каждого видно: голос, количество символов, когда создан.

## Счётчик использования

В Настройках есть счётчик, который покажет, сколько символов синтеза вы потратили в этом месяце из 200 000.

Счётчик обновляется автоматически после каждой озвучки и сбрасывается 1-го числа каждого месяца. Это просто подсказка — следить за лимитом, чтобы не упереться в него неожиданно.

## Резервное копирование и перенос

Глагол умеет сохранять всю библиотеку (документы + аудиофайлы) в один архив — удобно для бэкапа или переноса на другой компьютер.

**Создать копию:**
Настройки → **«Создать резервную копию»** → выберите папку. Получится один `.zip`-файл со всем содержимым.

**Восстановить / перенести на новый компьютер:**
1. На новом компьютере установите Глагол, настройте ключ SaluteSpeech.
2. Настройки → **«Восстановить из резервной копии»** → выберите ваш `.zip`.
3. Глагол покажет, что заменит, и попросит подтверждение.
4. После восстановления приложение перезапустится — вся библиотека на месте.

Перед восстановлением Глагол автоматически создаёт резервную копию текущего состояния — на случай, если что-то пойдёт не так.

## Если что-то не работает

**Озвучка не запускается / ошибка ключа**
Проверьте ключ в Настройках (кнопка «Проверить»). Убедитесь, что есть интернет. Проверьте счётчик — не закончился ли месячный лимит 200 000 символов.

**Установщик не запускается — предупреждение Windows**
Это SmartScreen. «Подробнее» → «Выполнить в любом случае». Подробнее в разделе «Установка».

**Голос звучит непривычно**
Попробуйте другой голос из шести — у каждого своя манера. Если озвучиваете не русский текст — это не сработает, Глагол только для русского.
